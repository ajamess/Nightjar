# Notification Sounds

This directory contains royalty-free notification sounds for Nightjar.

## Required Sound Files

The following MP3 files are required:

| File | Description | Suggested Source |
|------|-------------|------------------|
| `chime.mp3` | Gentle bell chime | [Pixabay](https://pixabay.com/sound-effects/) |
| `pop.mp3` | Soft bubble pop | [Mixkit](https://mixkit.co/free-sound-effects/) |
| `ding.mp3` | Single notification ding | [Freesound](https://freesound.org/) |
| `bell.mp3` | Clear bell tone | [Pixabay](https://pixabay.com/sound-effects/) |
| `subtle.mp3` | Soft whoosh | [Mixkit](https://mixkit.co/free-sound-effects/) |
| `ping.mp3` | Digital ping | [Freesound](https://freesound.org/) |
| `drop.mp3` | Water drop sound | [Pixabay](https://pixabay.com/sound-effects/) |
| `blip.mp3` | Retro blip | [Mixkit](https://mixkit.co/free-sound-effects/) |
| `tap.mp3` | Light tap | [Freesound](https://freesound.org/) |
| `sparkle.mp3` | Magical sparkle | [Pixabay](https://pixabay.com/sound-effects/) |

## Licensing

All sounds should be:
- Royalty-free
- CC0 or similar permissive license
- Free for commercial use

## Recommended Sources

1. **Pixabay Sound Effects** - https://pixabay.com/sound-effects/
   - All content is CC0 (public domain)
   - No attribution required

2. **Mixkit** - https://mixkit.co/free-sound-effects/
   - Free license for all sounds
   - No attribution required

3. **Freesound** - https://freesound.org/
   - Check individual license for each sound
   - Many CC0 sounds available

## File Requirements

- Format: MP3
- Duration: 0.5-2 seconds recommended
- File size: Keep under 50KB each for fast loading
- Quality: 128kbps is sufficient for notification sounds
